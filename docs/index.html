01

프로젝트 요약

"열은 어떻게 보이는가"를 센서로 직접 측정하고, 보정하고, 눈으로 보는 프로젝트입니다.

MLX90641 적외선 어레이 센서(16×12, 192픽셀)의 온도 프레임을 Nano ESP32가 I2C로 읽어 보정 계산 후 USB 시리얼로 PC에 전송하고, 브라우저의 Web Serial HTML 앱이 이를 실시간 열화상으로 렌더링합니다. 클라우드도 모델 학습도 없는 순수 측정·시각화 파이프라인이라, 흑체복사·적외선·방사율 단원과 곧장 연결됩니다.

교육 포인트 — 보간 OFF + 수동 스케일로 두면 손·금속·유리의 온도가 다르게 보이는 이유(방사율 차이)를 흑체복사·적외선 단원과 직접 연결해 토론할 수 있습니다.

03

준비물

하드웨어 (실물)

  • Arduino Nano ESP32ESP32-S3 · USB-C
  • Grove Thermal Imaging CameraMLX90641 · I2C 0x33 · 16×12
  • Grove 케이블SDA=A4 · SCL=A5
  • PCChrome / Edge (Web Serial)

소프트웨어 · 라이브러리

  • Arduino IDE2.x
  • MLX9064x_Universal (동봉)lib/*.zip · ESP32 패치 + 칩 자동판별 ★
  • Chrome / EdgeSafari·Firefox 불가
펌웨어는 헤더 하나 — 스케치는 래퍼 헤더 MLX9064x_Universal.h 하나만 include 하면 됩니다. 칩(90641/90640)은 EEPROM 0x240A bit6으로 자동 판별되어 부팅 시 INFO:CHIP=MLX90641,COLS=16,ROWS=12 한 줄로 통지되고, 픽셀 수(192/768)도 자동으로 맞춰집니다.
